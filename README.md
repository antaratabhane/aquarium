# Aquarium – Interactive Fish Exploration 🐠
**Aquarium** is an interactive web project built using **HTML, CSS, and Bootstrap 5** that simulates a realistic underwater environment. The interface presents a visually immersive aquarium scene where animated fish move naturally within a coral reef background, creating a dynamic and engaging user experience 🌊.

The project aims to recreate the feeling of observing a real aquarium through a digital interface. By combining visual design, animation, and interactive elements, the environment becomes more than just a static webpage—it becomes a small interactive ecosystem where users can observe and explore aquatic life. <br> <br>

## Overview 🌍

The project focuses on combining **front-end animation, user interaction, and visual design** to create a lifelike digital aquarium. Users can interact directly with the fish elements to reveal information, transforming a simple visual scene into an educational and exploratory interface.

Instead of simply displaying images, the aquarium environment is designed to feel alive. Fish continuously move through the underwater space, creating motion and depth within the interface. The interaction model allows users to discover information in a natural way, similar to observing and learning about fish in a real aquarium setting.

This approach demonstrates how modern front-end technologies can be used to build **interactive visual experiences** rather than traditional static webpages. <br> <br>

## Key Features ✨

### Realistic Aquarium Environment 🪸

The application displays a detailed underwater scene containing coral reefs, aquatic plants, and animated fish. The visual design aims to replicate the feel of a real aquarium.

Careful styling and positioning of elements help create a layered underwater environment that enhances realism. The combination of colors, background visuals, and aquatic elements gives the page the atmosphere of a natural marine habitat.

### Animated Fish Movement 🐟

Fish are animated to move across the screen, giving the environment a lively and natural appearance. The motion enhances immersion and simulates real aquatic behavior.

Continuous movement across the aquarium creates the illusion of an active underwater ecosystem. This animation helps prevent the interface from feeling static and encourages users to observe the screen as if watching a real aquarium tank.

### Interactive Fish Identification 🔍

Users can interact with individual fish to access information:

* **Single Click:** Displays the name of the selected fish.
* **Double Click:** Redirects the user to a dedicated page containing detailed information about that particular fish.

This interaction model enables both quick identification and deeper exploration. It allows users to quickly recognize fish species while also providing the opportunity to learn more about them through dedicated information pages.

This feature transforms the aquarium from a simple visual design into a **mini interactive learning experience**. <br> <br>

## Technologies Used 🛠️
